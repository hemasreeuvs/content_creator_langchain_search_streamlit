# 🌟 Content Creator Search with LangChain + Streamlit

This project demonstrates an **end-to-end LLM workflow** using **OpenAI**, **LangChain chains**, and **Streamlit**.  
It allows users to input a content creator’s name and retrieve structured information such as birth date and historical events around that time.

---

## 🚀 Features

- Search for any content creator by name  
- Retrieve their birth date and life details  
- Generate 5 major events around their birth year  
- Combines multiple chains with **SimpleSequentialChain** and **SequentialChain**  
- Conversation memory to retain chat history  
- Interactive **Streamlit** interface  

---

## 🛠️ Tech Stack

- Python  
- LangChain  
- OpenAI API  
- Streamlit
